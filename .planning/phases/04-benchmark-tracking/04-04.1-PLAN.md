Fix the CSS custom property typo at two locations in app/static/style.css:

Line 674 (.comparison-table th, .comparison-table td):
```
border-bottom: 1px solid var(----border, #2a2a2a);
```
Change to:
```
border-bottom: 1px solid var(--border, #2a2a2a);
```

Line 715 (.attribution-table th, .attribution-table td):
```
border-bottom: 1px solid var(----border, #2a2a2a);
```
Change to:
```
border-bottom: 1px solid var(--border, #2a2a2a);
```

The typo `----border` is four dashes which is invalid CSS custom property syntax.
The browser silently ignores this declaration and falls back to `#2a2a2a`.
Fixing to `--border` (correct two-dash syntax) ensures the CSS variable is used.
</action>
