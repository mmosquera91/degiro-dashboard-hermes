```python
def compute_portfolio_weights(positions: list[dict]) -> list[dict]:
    total_value = sum(p.get("current_value_eur", 0) or 0 for p in positions)
    if total_value == 0:
        for pos in positions:
            pos["weight"] = 0.0
        return positions
    for pos in positions:
        val = pos.get("current_value_eur", 0) or 0
        pos["weight"] = round((val / total_value) * 100, 2)
    return positions
```


The main `compute_health_alerts(portfolio: dict) -> list[dict]` function:
- Receives portfolio dict with positions, sector_breakdown, etf_allocation_pct, stock_allocation_pct
- Calls _check_concentration, _check_sector_weighting, _check_drawdown, _check_rebalancing
- Returns list of alert dicts (filtered to exclude None)
- Use `pos.get("weight") or 0` for null-safe weight access
- Use `pos.get("perf_ytd") or 0` for null-safe YTD access

_check_concentration:
- Iterate positions, check if weight > HEALTH_POSITION_THRESHOLD
- Return list of concentration alerts with triggering_positions populated

_check_sector_weighting:
- Iterate sector_breakdown dict, check if pct > HEALTH_SECTOR_THRESHOLD
- Return list of sector alerts with triggering_positions: None

_check_drawdown:
- Compute weighted average of perf_ytd: sum(perf_ytd * current_value_eur) / sum(current_value_eur)
- Guard total_value == 0 with early return None
- Return single drawdown alert dict or None

_check_rebalancing:
- Compute drift: abs(etf_pct - TARGET_ETF_PCT) and abs(stock_pct - TARGET_STOCK_PCT)
- Trigger if either drift > HEALTH_REBALANCE_THRESHOLD
- Return single rebalancing alert dict or None

Each alert dict follows D-01 schema with type, severity, message, current_value, threshold, triggering_positions.
